# Contributing

Urban Matrix Agentic AI OS is designed as a disciplined architecture, BIM, AI design, and automation system.

## Contribution Areas
- Agent definitions
- Skill files
- Prompt quality
- Dashboard UI/UX
- Safety controls
- Documentation
- Tests
- Deployment workflows

## Standards
- Keep secrets out of Git.
- Use structured outputs for agent-to-agent handoffs.
- Add approval gates for risky actions.
- Prefer small, reviewable changes.
- Document assumptions in architecture, cost, compliance, and BIM outputs.

## Pull Request Checklist
- Code runs locally.
- README/docs are updated where needed.
- New outputs are examples only, not private project data.
- Safety implications are described.
- No `.env` or API keys are committed.
